# Nota Técnica: Lanzamiento Moonshadow NVR v0.0.2 🌙🛡️

## 📝 Resumen del Lanzamiento
Esta versión marca la transición de un sistema de grabación tradicional a una plataforma de vigilancia inteligente con IA integrada. Se han resuelto errores críticos de estabilidad encontrados durante la fase de restauración y se ha optimizado el motor de ejecución para hardware moderno.

## 🚀 Nuevas Funcionalidades
- **Motor IA Universal:** Soporte nativo para NPU (Intel Core Ultra), GPU/iGPU (Vulkan), ARM ACL (Raspberry Pi 5) y NVIDIA (CUDA).
- **Reconocimiento Facial (Face ID):** Sistema de registro de identidades y base de datos de embeddings.
- **LPR Dinámico (Chile):** Reconocimiento de patentes optimizado para formatos chilenos con re-entrenamiento automatizado.
- **Mapas Térmicos:** Análisis de comportamiento basado en tiempo de permanencia (Dwell-Time).
- **HUD en Vivo:** Interfaz de visualización con indicadores de resolución, FPS, estado de grabación y salud del stream.

## 🛠️ Correcciones Técnicas (Fixes)
1. **Conflicto de Directorios:** Se modificó la validación `is_empty` para permitir que la base de datos y las grabaciones compartan el mismo volumen físico.
2. **Permisos de BD:** El instalador y el servidor ahora manejan correctamente los permisos de SQLite para evitar errores de "Read-only database".
3. **Robustez del Reset:** El comando de Factory Reset ahora detecta dinámicamente la ubicación del binario activo.
4. **TUI mejorada:** Feedback visual en tiempo real para operaciones de disco y configuración de red.

# 📦 Instrucciones de Instalación y Uso

## 1. Requisitos Previos
- Linux (Debian/Ubuntu, Arch, Fedora).
- Al menos 2GB de espacio libre para modelos de IA.
- (Opcional) Hardware compatible con OpenVINO o CUDA para máximo rendimiento.

## 2. Instalación Automática
```bash
git clone https://github.com/washaka81/Moonshadow-NVR.git
cd Moonshadow-NVR
sudo ./install.sh
```

## 3. Configuración Inicial
Lanza el asistente interactivo para añadir tus cámaras y activar la IA:
```bash
./configure-tui.sh
```
- Ve a **Cameras** -> **Add Camera**.
- Ve a **AI Settings** y activa:
  - `[Detect] Object Detection`
  - `[Face] Face Registration` (Opcional)

## 4. Ejecución
Para modo producción (como servicio):
```bash
sudo systemctl enable --now moonshadow-nvr
```
Para modo desarrollo/manual:
```bash
./start-server.sh
```

## 5. Uso de la Interfaz Web
Accede a `http://localhost:8080`
- **Usuario:** `admin` / **Password:** `admin`
- **Live:** Ver cámaras en tiempo real.
- **Archives:** Reproducir grabaciones históricas.
- **AI Events:** Filtrar por personas, vehículos o patentes detectadas.

# 🔒 Seguridad y GitHub (Best Practices)
Este repositorio está preparado para subirse a GitHub de forma segura:
- **.gitignore:** Configurado para excluir archivos `.db`, `.env`, secretos de sesión, modelos ONNX pesados y carpetas de desarrollo (`target/`, `node_modules/`).
- **Configuración:** No subir nunca archivos `config.toml` con contraseñas reales; usar `config.toml.example`.
- **Datos Sensibles:** Antes de subir, asegúrate de que la carpeta `models/face_data/registered/` esté vacía si ya has registrado personas reales.
